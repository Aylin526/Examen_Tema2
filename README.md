# 🧤 Juego de Portero - Canvas Game

Proyecto interactivo desarrollado con HTML, CSS y JavaScript utilizando Canvas API.  
El jugador asume el rol de un portero que debe atajar balones que se dirigen hacia la pantalla.

---

## 🎮 Vista del juego

El objetivo es hacer clic sobre los balones para atajarlos antes de que lleguen al jugador.

---

## ✨ Características principales

- ⚽ Balones con movimiento dinámico hacia el jugador
- 🧤 Sistema de atajadas con clic del mouse
- 💥 Efectos de partículas al detener balones
- 📈 Sistema de niveles progresivos
- ⚡ Aumento de dificultad automático
- 🏆 Sistema de récord (High Score) con `localStorage`
- 🎮 Pantalla de inicio con instrucciones
- 🖱️ Cursor personalizado (guante de portero)
- 🌄 Fondo tipo estadio con efecto visual tipo “glass”

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API
- Bootstrap 5
- LocalStorage
---
## 🧠 Mecánicas del juego
Cada balón atajado suma puntos
Cada 50 puntos sube el nivel
La velocidad aumenta progresivamente
Los balones se regeneran continuamente
El récord se guarda automáticamente en el navegador
